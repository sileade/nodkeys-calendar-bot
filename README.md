# Nodkeys Bot v9.11

Nodkeys Bot — это All-in-One AI-ассистент в Telegram, который заменяет 5 ключевых приложений для продуктивности и жизни: Calendar, Todoist, Notion, Trello и Zenmoney.

## 🌟 Ключевые возможности (Позиционирование "5 в 1")

1. **Умный календарь (Apple/Google Calendar)**
   - Полная синхронизация с iCloud
   - Управление встречами через естественный язык
   - Утренний брифинг

2. **Задачи и привычки (Todoist-killer)**
   - 4 списка задач (сегодня, неделя, месяц, когда-нибудь)
   - Трекер привычек с геймификацией и статистикой
   - Интеграция с Apple Reminders

3. **Заметки и база знаний (Notion-killer)**
   - Сохранение идей, ссылок, статей
   - Личный дневник
   - Интеграция с Apple Notes

4. **Проекты и Канбан (Trello-killer) — *NEW в v9.0***
   - Создание проектов и декомпозиция задач
   - Визуальная канбан-доска (To Do → In Progress → Done)
   - Управление приоритетами и прогрессом прямо в Telegram

5. **Финансовый трекер (Zenmoney-killer) — *NEW в v9.0***
   - Учет доходов и расходов по категориям
   - Установка месячных бюджетов с алертами
   - Генерация графиков (pie/bar) и финансовых отчетов

6. **Библиотека и Аудиокниги (Kindle/Bookmate)**
   - Поиск текстовых книг (Flibusta/Jackett)
   - Отправка EPUB прямо на Kindle
   - Поиск аудиокниг на RuTracker
   - Автоматическое скачивание и загрузка в S3 (параллельная загрузка, 8 потоков)
   - Встроенный аудиоплеер прямо в чате Telegram (Play/Prev/Next)
   - Автоматическая подгрузка следующей главы при нажатии Play
   - Веб-плеер (Mini App) с полным списком файлов
   - Кэширование на S3 для мгновенного повторного доступа

## 🚀 Как использовать новые функции

### Проекты (Trello)
Просто напишите боту:
- *"Создай проект Ремонт квартиры"*
- *"Добавь задачу: купить краску"*
- *"Покраска стен готова"*
Или используйте команду `/projects` для просмотра канбан-доски.

### Финансы (Zenmoney)
Просто напишите боту:
- *"Потратил 500 на обед"*
- *"Получил зарплату 150000"*
- *"Установи бюджет на еду 15000 в месяц"*
- *"Покажи расходы за месяц"*
Или используйте команду `/finance` для просмотра финансового дашборда.

### Аудиокниги (RuTracker)
Просто напишите боту:
- *"Найди аудиокнигу Мастер и Маргарита"*
- *"Скачай аудиокнигу Пелевина"*

Бот найдет книгу на RuTracker, скачает через qBittorrent, загрузит в S3 и предоставит:
- **Кнопки Play/Prev/Next** — слушайте прямо в чате Telegram
- **Автоподгрузка** — при нажатии Play приходит текущая + следующая глава
- **Веб-плеер** — полный список файлов с навигацией по главам
- **Кэш** — повторный запрос той же книги — мгновенный результат

## 🛠 Установка и запуск

1. Клонируйте репозиторий
2. Установите зависимости: `pip install -r requirements.txt`
3. Настройте переменные окружения в `.env` (TELEGRAM_TOKEN, ANTHROPIC_API_KEY и др.)
4. Запустите бота: `python bot.py`
